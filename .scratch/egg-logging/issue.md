## What to build
Develop a mobile-friendly web app for logging daily egg production. It should allow users to enter the weight and date of an egg laid by a specific chicken. It should enforce a limit of one egg per day per chicken.

## Acceptance criteria
- [ ] Mobile web app displays a list of chickens.
- [ ] Users can enter weight and date for an egg.
- [ ] "Quick Log" button defaults to the current date and the most recently used chicken.
- [ ] System prevents logging multiple eggs for the same chicken on the same day.
