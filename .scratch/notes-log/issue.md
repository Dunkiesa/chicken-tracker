# Chronological Notes Log

## What to build

A chronological notes log on each chicken record for tracking events such as vet visits and medications. Users can add, edit, and delete dated notes. The profile displays the notes in chronological order (newest first).

End-to-end: the UI for composing and managing notes, the API persisting note text with timestamps against the chicken, and the ordered display.

## Acceptance criteria

- [x] A user can add a dated note to a chicken
- [x] A user can edit an existing note
- [x] A user can delete a note
- [x] The chicken profile lists notes in chronological order
- [x] Automated tests cover add, edit, and delete of notes
