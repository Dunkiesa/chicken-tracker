## What to build
Create a desktop tool for enrolling new chickens. This includes a form to input Name, Sex (Hen/Rooster/Unknown), Breed (dynamic list), Origin Source (dynamic list), and Acquisition Type (dynamic list). It should also support initial photo uploads with free-form descriptions.

## Acceptance criteria
- [ ] Desktop tool allows creating a new chicken with all required fields.
- [ ] Breed, Origin, and Acquisition fields update the dynamic lists in the database.
- [ ] Users can upload photos with descriptions.
- [ ] Unique IDs are automatically assigned to each chicken.
