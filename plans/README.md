- **`getUserByEmail` called twice per auth flow**: Negligible for ~5 users. Not worth a plan.
- **Hardcoded Southern Hemisphere season mapping**: Documented design choice per `CONTEXT.md`.
- **All pages client-side rendered**: ADR 0005 documents the always-online-over-LAN design.
- **CSRF not mitigated**: LAN-local with trusted user base. Acceptable risk.
- **No rate limiting**: Same reasoning as CSRF.

- **Photo path traversal** — `src/app/api/photos/[filename]/route.ts:30-43` already has path traversal guards (checks `..`, `/`, `\`, resolves path). By-design protection.
- **CSV export URL injection** — Uses `window.open` with relative URL, same-origin only. Acceptable in LAN deployment context.
- **Unused testing-library deps** — `@testing-library/react` and `@testing-library/jest-dom` are unused but become actionable via Plan 025 which adds component tests that use them.
- **Suspense wrapper in log-egg page** — Pre-existing dead code, negligible impact. Not worth a plan.
- **Non-null assertion on productionData** (`src/app/page.tsx:393`) — Brittle but functional. Deferred to Plan 031's useMemo refactor which incidentally removes the `!`.
